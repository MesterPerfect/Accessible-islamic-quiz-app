# تطبيق أسئلة إسلامية (Islamic Quiz App) 🌙

تطبيق تفاعلي مبني باستخدام **React Native** و **Expo**، يهدف إلى تقويم مستوى المستخدمين في العلوم الشرعية وتطوير حصيلتهم العلمية في مجالات متنوعة (التفسير، العقيدة، الحديث، الفقه، التاريخ، واللغة العربية) بأسلوب ممتع ومتاح للجميع. جميع الأسئلة مستمدة من موقع الدرر السنية الموثوق.

## 🚀 المميزات الرئيسية

* **دعم إمكانية الوصول (Accessibility):** التطبيق متوافق بالكامل بنسبة 100% مع قارئات الشاشة (Screen Readers) لضمان تجربة استخدام سلسة للمكفوفين وضعاف البصر.
* **نظام التدرج (Gamification):** مستويات متدرجة الصعوبة، محاولات (Lives)، ومؤقت زمني لكل سؤال مع نظام تسجيل النقاط.
* **التخصيص والمظهر:** يدعم 3 أوضاع مختلفة (الوضع الفاتح، الوضع المظلم، ووضع التباين العالي).
* **مؤثرات تفاعلية:** مؤثرات صوتية عند الإجابة واهتزازات تفاعلية (Haptics) تزيد من حماس اللاعب.
* **حفظ التقدم (Local Storage):** يتم حفظ الإنجازات والمستويات المفتوحة والإحصائيات محلياً على جهاز المستخدم.
* **مراجعة الأخطاء:** شاشة مخصصة لمراجعة الأسئلة الخاطئة وتصحيحها بعد انتهاء المستوى.

## 🛠️ التقنيات المستخدمة

* React Native
* Expo & EAS Build
* Context API (State Management)
* AsyncStorage (Local Data Persistence)
* Expo AV & Expo Haptics (Audio & Sensory Feedback)

## ⚙️ طريقة التشغيل محلياً (Installation)

1. Clone the repository:
```bash
git clone https://github.com/mesterPerfect/Accessible-islamic-quiz-app.git
````

2.  Navigate to the project directory:
```bash
cd Accessible-islamic-quiz-app
```

3.  Install dependencies:
```bash
npm install
```

4.  Start the Expo server:
```bash
npx expo start
```
